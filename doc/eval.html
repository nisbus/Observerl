Description

Takes a string in the format "fun(X) X+1 end" and returns it as an executable fun. Also supports binary formatted string fun and an actual fun.

make_fun/1

make_fun(Fun::undefined | function() | binary() | string()) -> function()

Creates a function from a string or binary. If input is function it just returns the function.

make_funs(Funs::undefined | [string()] | [function()] | [binary()]) -> [function()]

Creates a list of functions from a list of strings or binary. If input is a function list it just returns the function list.
